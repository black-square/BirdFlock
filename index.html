Rapid 2D Game Prototyping Engine

mango2d is a cross platform 2D game engine specially designed for fast 2D games prototyping. It uses C++11 and BOOST libraries to provide powerful yet easy-to-use features out of the box. It is highly modulized and based on SDL Framework, but uses only OpenGL for rendering.

Features

  • Cross platform developing on Windows and Mac OS
  • Mit licence
  • Components oriented: you can use only modules that you want
  • Logging, common macroses, easy string formating (boost::format based)
  • Powerful Point, Rect, Size and Color classes that interact with each other
  • Many interpolation functions including physically reliable spring formula
  • Intrusive lists for fast collection manipulating without memory allocations
  • Sprites loading from common formats (PNG, JPG, TIFF) and drawing with scale, transparency and color blending. One texture can contain many frames
  • Fonts loading from TTF format
  • Sound engine
  • Particles engine
  • GUI framework: collection of GUI widgets and GUI states for rapid interface development
  • Special boost::shared_ptr delete strategy that keeps object destruction until Update() method finishes, but weak pointers become expired immediately. It simplifies AI logic
  • etc.

For more information and documentation, please have a look at the project page
